Application Modules

Agents & Functions

  • interpret — Intent classifier (Strands + Bedrock Claude). Classifies user input into new_feature, modify_graph, generate_code, or explain. Keyword fallback on LLM failure.
  • architect — AWS Solutions Architect agent (Strands + Bedrock Claude). Generates/modifies React Flow node graphs with auto-positioning across 7 column types.
  • security_review — Security gate before code generation. Includes security autofix module.
  • cdk_specialist — Multi-format IaC code generator: CDK TypeScript (LLM via Strands), CDK Python, CloudFormation YAML, and Terraform HCL (template-based).
  • react_specialist — Generates React + Cloudscape frontend code for the designed architecture.
  • get_execution — Polls Step Functions execution status. Lightweight (256 MB, 30s timeout, shared layer only).
Test coverage — functionspytest + Vitest

Lambda Layers & Runtime

  • SharedLayer — pydantic-settings, AWS Powertools, X-Ray SDK
  • AgentsLayer — strands-agents core for Bedrock model invocation
  • Runtime: Python 3.12, managed with uv
  • Default: 512 MB memory, 5 min timeout, X-Ray active tracing
  • CloudWatch alarms per function: error count (threshold 1) and p99 duration (80% of timeout)
  • Cross-region Bedrock: us-east-1, us-east-2, us-west-2 for us.* inference profiles
Test coverage — layers/backendpytest

Infrastructure (CDK v2)

  • DatabaseStack — Cognito User Pool + Identity Pool, DynamoDB sessions table, S3 hosting bucket, CloudFront distribution, SNS alarm topic, monthly budget ($25)
  • FunctionsStack — 6 Lambda functions, 2 PythonLayerVersions, Bedrock IAM policies, CloudWatch error + p99 duration alarms
  • WorkflowStack — Step Functions state machine, Cognito identity pool role bindings, cross-table DynamoDB access for planner handoff
Test coverage — infraJest / CDK assertions

Frontend (React 19 SPA)

  • Canvas.tsx — React Flow architecture canvas with 65 custom AWS node types, auto-positioned across 7 columns
  • Chat.tsx — AI chat panel + code generation trigger UI
  • GeneratedCodeModal.tsx — Displays generated IaC code (CDK/CFN/TF/Python-CDK)
  • PlannerNotification.tsx — DynamoDB handoff toast from Project Planner AI
  • PlannerRefineButton.tsx — Refine imported planner data
  • security-autofix.ts — Client-side security analysis (no AWS call needed)
  • State: Zustand stores (chat, graph). Auth: Amplify Authenticator
Test coverage — webVitest
