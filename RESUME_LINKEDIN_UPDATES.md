**"Why did you build this?"**
"Two reasons: First, to demonstrate rapid learning of unfamiliar frameworks. Second, to 
showcase framework diversity - I now have experience with both AWS-native (Step Functions) 
and framework-based (LangGraph) orchestration, and I can articulate when to use each."

**"What's the difference from Resume Tailor AI?"**
"Resume Tailor AI uses AWS Step Functions for deterministic workflows - great for 
predictable, AWS-native flows. Scaffold AI uses LangGraph for dynamic AI routing - better 
for conversational AI where routing depends on LLM responses. Both have production patterns 
like security gates and rate limiting, but different orchestration approaches."

**"What did you learn?"**
"Three key things: First, LangGraph's built-in state management is much simpler than 
managing S3/DynamoDB. Second, local development is 10x faster without AWS deployment. 
Third, dynamic routing based on LLM content is powerful but requires careful error handling."
